# ☕️ ms-pharmacys-turn

> ℹ️ rest api to organize and get nearest open pharmacy

## ☝️ Dependencies

* Java 11
* Gradle (Wrapper included)
* Docker (if you want to run integration test)

## 🏁 Install and Run

```bash
 ./gradlew bootRun
```
